# 일정 드래그 도구 색상 대비 개선 설계

## 목표

관리자 수업 일정의 월별 수업일 도구에서 보라색과 하늘색을 한눈에 구분할 수 있게 한다.

## 범위

- `monthColor()`의 첫 두 월 색상 세트를 교체한다.
- 도구 버튼과 달력 수업일 셀이 같은 색상 세트를 사용하므로 두 화면에 일관되게 반영한다.
- 데이터 모델, 선택 동작, 드래그 동작 및 다른 월 색상은 변경하지 않는다.

## 색상

| 월 순서 | 기존 | 변경 |
| --- | --- | --- |
| 첫 번째 | 연보라 | 옅은 라벤더 배경 + 진한 자주색 테두리 |
| 두 번째 | 연하늘 | 옅은 민트 배경 + 진한 청록색 테두리 |

색상은 배경뿐 아니라 테두리에도 충분한 색조 차이를 두어, 인접하거나 겹쳐 보이는 경우에도 구분한다.

## 검증

- 기존 관리자 일정 Playwright 명세에 두 도구의 분리된 클래스 색상 계약을 추가한다.
- 테스트가 기존 색상에서 실패하는 것을 확인한 뒤 색상 토큰을 최소 변경으로 교체한다.
- 변경된 명세와 TypeScript 검사를 실행한다.
