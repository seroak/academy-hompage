# 수학교육 과정 카드 레이아웃 설계

## 목표

`/courses`의 세 과정 카드에서 사진과 텍스트의 시작점이 어긋나 보이는 문제를 해결한다. 학부모가 과정을 빠르게 비교할 수 있도록, 모든 카드의 사진과 정보 묶음을 같은 순서와 기준선으로 배치한다.

## 범위

`frontend/src/components/math-curriculum/ProgramCards.tsx`의 카드 내부 레이아웃만 수정한다. 과정 데이터, 문구, 링크, 색상 체계는 변경하지 않는다.

## 레이아웃

1. 카드 상단에 영문 과정명과 한글 과정명을 둔다.
2. 그 아래에 고정 종횡비의 수업 사진을 배치한다. 세 카드의 사진 시작선은 동일하다.
3. 사진 아래에 과정 요약, 핵심 학습 항목, 대상 연령, 상담 버튼을 순서대로 둔다.
4. 카드 컨테이너는 세 열에서 같은 높이로 유지하며, 상담 버튼은 카드 하단에 정렬한다.
5. 좁은 화면에서는 카드가 한 열로 쌓이되, 정보 순서를 그대로 유지한다.

## 검증

- 데스크톱 너비에서 세 카드의 사진 상단과 핵심 정보 영역이 정렬된다.
- 모바일에서 콘텐츠가 겹치거나 잘리지 않는다.
- TypeScript 검사와 프로덕션 빌드가 통과한다.
